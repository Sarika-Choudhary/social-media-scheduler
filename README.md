# Social Media Scheduler

## Description
A MERN Stack application that allows users to create, schedule, and manage social media posts.

## Features
- User Authentication (JWT)
- Create/Edit/Delete Posts
- Schedule Posts
- Cron Job
- Admin Panel
- MongoDB Atlas
- React Frontend
- Express Backend

## Tech Stack
- React
- Node.js
- Express
- MongoDB Atlas
- JWT
- Multer
- Node-Cron
- Bootstrap

## Installation

### Backend
cd backend
npm install
npm run dev

### Frontend
cd frontend
npm install
npm run dev

## Deployment
Frontend: Vercel

Backend: Render

Database: MongoDB Atlas
